# Vite + React + Tailwind Starter

A minimal, production-ready starter using **Vite**, **React**, and **Tailwind CSS**.

## Quickstart

```bash
npm install
npm run dev
# or
pnpm install && pnpm dev
# or
yarn && yarn dev
```

## Build
```bash
npm run build
npm run preview
```

## What's inside
- Vite + React (HMR)
- Tailwind CSS (PostCSS)
- Minimal, clean UI in `App.jsx`
